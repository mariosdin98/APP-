# StoryStream

StoryStream is a fuller TikTok-inspired discovery app concept for books and movies.

## What is included

- Login and registration flow with local account storage
- Seeded local database with many books and movies across multiple genres
- Personalized vertical feed with category and genre filters
- Save, like, and share interactions
- User-owned custom uploads that get added into the local database
- Responsive glassmorphism UI with animated media cards
